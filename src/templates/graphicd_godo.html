GODO Website

Client | Christian Fumz

Godo is concept created by Christian Fumz an creative project manager in Byens Rum. The concept was made for a proposal for the Aalborg Kommune (the local city government). The concept of GODO is a simple application website that could be used for getting support for varous creative projects. There is some basic branding included in the project.

The applications can be used to submit, get feedback and manage local activities and projects. Included in the application is a map with various locations to choose from and a calendar to keep track that events so they would not overlao. Some custom icons were developed for the website.
